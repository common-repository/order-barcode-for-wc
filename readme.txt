== Description ==

Order Barcode for WC is a plugin you can add for your ecommerce site. It can track product information through scans. It generates a different barcode for each individual order placed on your woocommerce site. These barcodes can be used on packing slips, order confirmation emails and invoices or anything that benefits from having a scannable barcode attached to it. It also provides print order details facility after woocommerce porduct order.

> **Note: This plugin requires PHP 7.0 or higher to be activated.**

* Go to your Dashboard after installation and navigate to "Order Barcode for WC >> Settings" to configure this Barcode.
